
# 🛡️ System Sentinel Pro

**Mobile Network Diagnostics & Intelligence Suite**

## 📖 Project Overview
System Sentinel is a Python-based toolkit designed for real-time network auditing from a mobile node. This project demonstrates the ability to capture geolocation data, test handshake latency, and verify ISP routing integrity.

### 🚀 Key Features
- **Geolocation Mapping:** Identifies external IP, City, and Provider (ISP).
- **Latency Benchmarking:** High-precision TCP handshake timing.
- **Network Recon:** Automated discovery of local network devices.
- **Log Management:** Automated backup and export of audit data.

## 📊 Proof of Concept
I have successfully audited multiple network environments:
- **Kelowna, Canada (TELUS):** Verified high-speed sub-30ms routing.
- **Seattle, USA (Datacamp):** Identified data center exit nodes.
